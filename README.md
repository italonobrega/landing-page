# 🚀 TaskFlow — Landing Page

Landing page de um SaaS fictício de gestão de tarefas, desenvolvida com HTML, CSS e JavaScript puro.

**[👉 Clique aqui para acessar o site](https://italonobrega.github.io/landing-page/)**

## ✨ Funcionalidades

- Hero com imagem animada (floating)
- Seção Sobre com layout dois colunas
- Cards de recursos com efeito hover
- Formulário de contato estilizado
- Menu mobile com animação e troca de ícone
- Layout 100% responsivo
